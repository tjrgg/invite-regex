> Regular expression for matching [Discord](https://discordapp.com) invite URLs

## Install

```sh
$ npm install invite-regex
```

## Usage

```js
const inviteRegex = require('invite-regex');

inviteRegex().test('https://discord.gg/2JY79nd');
// => true
```
